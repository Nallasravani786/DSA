### I N T U I T I O N 
Rules:

If number is even → divide by 2
(binary → remove last 0)

If number is odd → add 1
(binary → handle carry like normal addition)

We process from right to left and simulate the carry.

###### C O D E
```cpp
int numSteps(string s) {
    int n = s.size();
    int steps = 0;
    int carry = 0;

    // Start from rightmost bit down to index 1
    for (int i = n - 1; i > 0; i--) {
        int bit = (s[i] - '0') + carry;

        if (bit == 1) {
            // odd → add 1 (carry generated) + divide by 2
            steps += 2;
            carry = 1;
        } else {
            // even → just divide by 2
            steps += 1;
        }
    }

    // If carry remains at MSB
    return steps + carry;
    }

```
